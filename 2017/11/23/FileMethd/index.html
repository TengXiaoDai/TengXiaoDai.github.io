Python文件操作

发表于 2017-11-23 | 分类于 后端

1.open()
参数
a.文件名(绝对路径)
b.模式
c.编码
//默认模式是只读模式(r)
打开的几种模式:
r:只读模式(不存在则报错)
w:只写模式(不可读,不存在则创建,存在则清空内容)
x:只写模式(不可读,不存在则创建,存在则报错)
a:追加模式(可读,不存在则创建,存在只追加内容)

2.以字节的方式打开
a.rb
b.xb
c.wb
d.ab

3.+
//表示可读可写

4.seek()
//调整指针的位置
seek(num)

5.tell()
//获取指针的位置
tell(num)

6.flush()
//主动的将内存中的信息保存到硬盘中
7.readline()
//读取一行

8.truncate()
//截取指针之前的数据，(依赖于指针)
